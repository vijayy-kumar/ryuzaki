# L Lawliet — Detective Portfolio

> “When solving a case, I don’t act on emotion. I act on facts.”

A dark cinematic portfolio inspired by **L Lawliet (Ryuzaki)** from *Death Note*.

Built with a psychological noir aesthetic, this project combines:
- detective-inspired UI
- strategic investigation visuals
- premium animated themes
- immersive dark atmosphere
- responsive modern layout

The portfolio is designed to feel like a classified investigation system rather than a traditional website.

---

## Features

- Cinematic hero section
- Multiple premium themes
- Animated hover effects
- Investigation-style UI
- Responsive design
- Dark psychological aesthetic
- Glassmorphism cards
- Interactive sections
- Custom cursor effects
- Formspree contact integration

---

## Philosophy

This portfolio was designed around:
